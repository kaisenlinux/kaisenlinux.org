    NAME

       restic-key-list - List keys (passwords)

    SYNOPSIS

       restic key list [flags]

    DESCRIPTION

       The  "list"  sub-command lists all the keys (passwords) associated with the repository.  Returns the key ID, username, hostname, created time and if it's the current key being used to
   access the repository.

    EXIT STATUS

       Exit status is 0 if the command was successful.  Exit status is 1 if there was any error.  Exit status is 10 if the repository does not exist.  Exit status is 11 if the repository  is
   already locked.  Exit status is 12 if the password is incorrect.

    OPTIONS

       -h, --help[=false]      help for list
