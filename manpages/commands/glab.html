    NAME

       glab - A GitLab CLI tool.

    SYNOPSIS

       glab   [flags]

    DESCRIPTION

       GLab is an open source GitLab CLI tool that brings GitLab to your command line.

    OPTIONS

       --help[=false]      Show help for this command.

   -v, --version[=false]      show glab version information
